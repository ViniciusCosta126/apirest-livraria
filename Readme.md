# Livraria - Api Rest

### Rodando localmente

- Clone o repositorio

```bash
git clone https://github.com/ViniciusCosta126/apirest-livraria.git
```

- Crie uma virutalenv

```bash
python -m venv env
```

- Ative a virtualenv

```bash
.\env\Scripts\activate
```

- Instale as dependencias

```bash
pip install -r requirements.txt
```
